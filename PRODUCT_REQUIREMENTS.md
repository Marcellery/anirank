AniRank Product Requirements

Core idea:
Users rank anime by choosing which show is better in head‑to‑head battles.

Target users:
Anime fans who currently use AniList or MyAnimeList.

Key features:
- onboarding
- anime database
- anime battles
- ranking algorithm
- Top 25 ranking
- watch progress
- share ranking cards
- friend comparison
- global rankings

Out of scope for v1:
- manga ranking
- chat
- forums
